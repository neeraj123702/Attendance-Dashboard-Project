# Attendance-Dashboard-Project
Attendance Analysis Dashboard using Excel and Power BI
📌 Attendance Analysis Dashboard (Excel + Power BI)
📊 Project Overview

This project is a complete Attendance Analysis System developed using Microsoft Excel for data management and Power BI for interactive dashboard visualization. The objective of this project is to track, analyze, and visualize attendance records efficiently while providing meaningful insights through dynamic reporting.

The system allows users to monitor attendance percentages, present/absent trends, and overall performance using automated calculations and visual analytics.

🎯 Project Objective

To automate attendance tracking

To calculate attendance percentage dynamically

To visualize attendance trends using dashboards

To provide clear and interactive reporting

To reduce manual calculation errors

🧩 Project Components
1️⃣ Excel Data Sheet

The Excel file serves as the primary data source.

Features implemented in Excel:

Structured attendance database

Automated attendance percentage calculation

Use of formulas such as:

COUNTIF

SUM

IF

Percentage calculations

Clean and organized tabular format

Ready-to-connect data source for Power BI

The Excel sheet ensures data accuracy and consistency before visualization.

2️⃣ Power BI Dashboard

The Power BI file (.pbix) is connected to the Excel data source and provides an interactive dashboard.

Dashboard features include:

Attendance percentage KPI cards

Present vs Absent comparison charts

Trend analysis using bar and column charts

Interactive slicers for filtering data

Dynamic visual updates based on user selection

DAX measures were used to calculate attendance metrics dynamically and improve analytical flexibility.

📈 Key Insights Generated

Present vs Absent ratio

Attendance trend analysis

Performance monitoring through KPIs
